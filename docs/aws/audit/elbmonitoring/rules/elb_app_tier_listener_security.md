### More Info:
Your app-tier Elastic Load Balancer (ELB) listeners should be using the HTTPS/SSL protocol to encrypt the communication between your application clients and the load balancer.

### Risk Level
High

### Address
Security

### Compliance Standards
PCIDSS, APRA, MAS
