### More Info:
AWS S3 buckets should not allow READ_ACP access to AWS authenticated users using ACLs in order to protect against unauthorized access.

### Risk Level
Medium

### Address
Security

### Compliance Standards
PCIDSS, APRA, MAS, NIST
